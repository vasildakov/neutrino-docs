# User Data Deletion Instructions (Facebook/Meta)

**Last updated:** 2026-03-05

This page explains how to request deletion of your Neutrino data, including data associated with Facebook/Meta Login.

## 1. Request deletion from within Facebook
If you used Facebook Login, you can request deletion via Facebook:

1. Go to your Facebook settings.
2. Navigate to **Apps and Websites**.
3. Find **Neutrino**.
4. Choose **Remove** or **Request deletion** (depending on Facebook UI).

Facebook will send us a deletion request. Once processed, we will remove the relevant data from Neutrino.

## 2. Request deletion by contacting us
You can also request deletion directly by email:

**privacy@neutrino.bg**

Please include:
- The email address associated with your Neutrino account (if available)
- The login provider used (Facebook/Google/X/Twitter)
- Any additional information that helps us identify your account

## 3. Deletion status page
If Facebook provides you a confirmation code, you may check the status here:

**Data Deletion Status:** `data-deletion-status.md`

## 4. What data is deleted
When a deletion request is processed, we remove:
- Your linked social login identifier (e.g. Facebook user ID)
- Your Neutrino account data (unless we must retain some data for legal/security reasons)

If we must retain limited data for legal or security purposes, we will minimize it and restrict access.

## 5. Timeframe
We aim to process deletion requests within a reasonable timeframe.
